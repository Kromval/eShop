### Для пользователя
```plantuml
@startuml
|Покупатель|
:Войти в систему (email, пароль);
if (Авторизация успешна?) then (да)
  :Просмотреть каталог товаров;
  :Выбрать товар;
  :Добавить товар в корзину;
  while (Добавить ещё товары?) is (да)
    :Выбрать другой товар;
    :Добавить товар в корзину;
  endwhile (нет)
  :Перейти к оформлению заказа;
  :Указать способ доставки и оплаты;
  :Подтвердить заказ;
  :Получить подтверждение;
else (нет)
  :Зарегистрироваться;
  :Повторить попытку входа;
endif
|Покупатель|
@enduml
```
